# Tech Stack
* **Architecture:** Microservices, Event-Driven
* **Frontend:** Angular
* **Backend:** Java, Spring Boot
* **Orchestration:** Kubernetes
* **Service Mesh:** Istio
* **Monitor Tools:** Kiali, Jaeger, EFK stack, Prometheus, Grafana
* **Messaging:** Apache Kafka
* **Change Data Capture:** Debezium
* **Database:** PostgreSQL
