# Scaling

## Vertical Scaling
This means getting bigger machine.

## Horizontal Scaling
Having multiple things capable of doing the same work.

## Database replication
Master database supports write operations. Slave database supports read operations.

## Database partitioning
Dividing work based on some attribute of data.

## Functional Decomposition
Separation of work based on the type - microservice decomposition.

## Caching
Temporary storage area that stores the result of expensive responses or frequently accessed data in memory.

## Content Delivery Network (CDN)
Host static assets in CDN.

## Data center
Support multiple data centers.
